
Name
====

lazyCounterJS

## Description
数値をゆっくりと変化させるjQueryプラグインです。
## Demo
![無題](https://dayservice-navi.net/list_company.php?prefCode=13)
## Requirement
jQuery
## Usage
数値を配置したいタグで実行
$("#tag_name").lazyCounter({
    oldNumber : oldNumber,
    newNumber : newNumber,
    dulation  : dulation
});
