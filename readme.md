
Name
====

lazyCounterJS

## Description
既存の数値を目的の数値までゆっくりと変化させるjQueryプラグインです。
## Demo
https://dayservice-navi.net/list_company.php?prefCode=13
## Requirement
jQuery
## Usage
数値を配置したいタグで実行
$("#tag_name").lazyCounter({
    oldNumber : oldNumber,
    newNumber : newNumber,
    dulation  : dulation
});
